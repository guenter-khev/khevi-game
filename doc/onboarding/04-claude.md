# 4. Claude Haiku 4.5 in VS Code

[← Onboarding-Übersicht](index.md) | [Hauptseite](../index.md)

---

## 4.1 Copilot Chat installieren

1. **Strg+Shift+X** → nach **„GitHub Copilot Chat"** suchen (Publisher: GitHub)
2. Installieren → **VS Code neu starten**

## 4.2 Anmelden

1. **Strg+Shift+P** → **„GitHub Copilot: Sign in"**
2. GitHub-Konto mit Copilot-Zugang verwenden

## 4.3 Modell wählen

1. Chat öffnen mit **Strg+L**
2. Modell-Dropdown → **Claude Haiku 4.5** wählen
3. Im Chat: `@` zeigt verfügbare Dateien (z.B. `@doc/index.md`)
