# 6. 🛠 Probleme & Lösungen

[← Onboarding-Übersicht](index.md) | [Hauptseite](../index.md)

---

| Problem | Lösung |
|---|---|
| `git` nicht erkannt | Git neu installieren + Windows neu starten |
| Keine Indexierung | **Strg+Shift+P** → „Reload Window" |
| Copilot Chat geht nicht | Anmeldung prüfen → https://github.com/settings/copilot |
| Chat-Fenster fehlt | **Strg+L** oder Copilot-Icon in Seitenleiste |
| Kein Copilot-Zugang | Student-Pack beantragen → [3. GitHub Copilot](03-copilot.md) |
