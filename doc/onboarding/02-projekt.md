## 2.1 Projekt klonen

Im VS Code Terminal:
```terminal
cd C:\github
git clone https://github.com/guenter-khev/khevi-game.git
cd khevi-game
```

Das Projekt liegt jetzt in `C:\github\khevi-game\`.

## 2.2 Projekt in VS Code öffnen

1. **File > Open Folder** → `C:\github\khevi-game` wählen
2. „Do you trust the authors?" → **Ja**
3. Warten, bis Indexierung fertig ist (rechts unten)
